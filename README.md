## 概要

YMM4上で動作するリアルタイムチャットツールです。Supabaseをバックエンドに使用し、ルームを作成して他のユーザーとメッセージをやり取りできます。

## 機能

- ルーム作成と参加（ルーム名は任意）
- ルーム一覧の表示と検索
- リアルタイムメッセージ送受信（WebSocket）
- ユーザー名の保存（次回起動時に自動入力）
- メッセージ履歴の自動保存（サーバーに保存）
- スパム対策
  - 1メッセージ140文字まで
  - 同一メッセージの連続送信禁止（30秒間隔）
  - 同一URLの連続送信禁止（30秒間隔）
  - Google Safe Browsing APIによる危険URLのブロック
- 利用規約とプライバシーポリシーの同意（JSONで保存）

## 動作環境

- ゆっくりMovieMaker4 v4.53.0.4以降（Lite版および通常版）
- .NET 10.0.9以降

## インストール方法

1. YMM4を終了する。
2. 配布されている.ymmeファイル（またはZIP）をダブルクリックしてインストールする。  
   または、手動で`YMM4ChatPlugin.dll`を（例: `C:\YMM4\user\plugin\`）にコピーする。
3. YMM4を起動する。

## 使い方

1. YMM4の「ツール」メニューから「YMM4チャット」を選択する。
2. 初回起動時に利用規約とプライバシーポリシーが表示される。両方のリンクをクリックして内容を確認し、同意チェックボックスをオンにする。
3. ユーザー名を入力する（匿名可）。
4. 左側のルーム一覧から既存のルームを選択するか、「新しいルーム」ボタンを押してルームを作成する。
5. 下部の入力欄にメッセージを入力し「送信」ボタンを押すと、同じルームの他の参加者にメッセージが届く。

## 注意事項

- メッセージは送信後に削除できません。個人情報や重要な情報は送信しないでください。
- ルームの削除や名前変更はユーザーはできません。開発者に問い合わせるか、Supabaseのダッシュボードから直接編集する必要があります。
- 本プラグインは開発者のSupabaseプロジェクトを使用しています。無料枠の制限（月間リクエスト数など）があるため、大量のメッセージを送信しないでください。
- 開発者はシステム保守のためにログを確認することがあります。
- Safe Browsing APIによるURLチェックを行います。APIキーは開発者のものを使用しています。

## アンインストール方法

YMM4の設定→プラグインからプラグインを無効にするか、（例: `C:\YMM4\user\plugin\`）から`YMM4ChatPlugin.dll`を削除してください。  
設定ファイル（`Agreement.json`）はプラグインフォルダまたは`%LocalAppData%\YMM4ChatPlugin\`に残ります。必要に応じて手動で削除してください。
