# YMM4ChatPlugin プライバシーポリシー

開発者は、本プラグインの利用にあたり、利用者の情報の取り扱いについて以下の通り定めます。

### 1. 取得する情報
本プラグインの機能提供および運営のため、以下の情報を取得・保存します。
- **ユーザー名**: 利用者が任意に設定する名称
- **メッセージ内容**: 送信されたテキストデータ
- **送信日時**: メッセージが送信された時刻
- **ルームID**: チャットルームを識別するための識別子
- **IPアドレスおよびアクセスログ**: サーバーへの接続時に自動的に記録される情報

### 2. データの管理と保存先
取得した情報は、開発者が利用する外部クラウドサービス「Supabase」（運営会社：Supabase, Inc. / サーバーの所在国：日本）のデータベースに保存され、厳重に管理されます。

### 3. 利用目的
取得した情報は、以下の目的の範囲内でのみ利用します。
- 本プラグインにおけるチャット機能の提供・画面への表示
- システムの保守、バグ対応、およびトラブルシューティング
- 利用規約に違反する行為（スパム、誹謗中傷等）の監視・抑止
- サービス改善のための利用状況の分析（個人を特定できない統計データとしての利用）

### 4. 個人情報の第三者提供および委託
1. 開発者は、法令に基づく場合を除き、取得した情報を本人の同意なく第三者に提供しません。
2. 本プラグインの機能提供にあたり、データの保存・処理を外部サービス（Supabase等）に委託しています。これは適切なサービス提供のために必要な範囲であり、第三者への無断提供には該当しません。

### 5. データの保持期間
1. 送信されたメッセージおよびそれに付随するデータは、サーバーの容量管理やプライバシー保護のため、一定期間経過後に自動的または手動で削除される場合があります。
2. 利用者は、パブリックチャットの特性上、一度送信したメッセージを原則として利用者自身で削除・編集できない場合があることを予め了承するものとします。

### 6. 安全管理措置と免責事項
1. 開発者は、データの漏洩や不正アクセスを防止するため、クラウドサービスのセキュリティ機能（RLS等のアクセス制御）を利用した安全管理に努めます。
2. 適切な安全対策を講じていたにもかかわらず発生した、予期せぬセキュリティ侵害、サーバー攻撃、データ漏洩等によって生じた損害について、開発者は一切の責任を負いません（開発者に故意または重大な過失がある場合を除きます）。
