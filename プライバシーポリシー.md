# プライバシーポリシー（YMM4チャットプラグイン）

開発者は、本プラグインの利用に際して以下の情報を収集・保存することがあります。

## 1. 収集する情報

- **ユーザー名**：ユーザーが任意に設定する名称（匿名可能）
- **メッセージの内容**：送信されたテキスト
- **送信日時**：メッセージのタイムスタンプ
- **ルームID**：自動生成される英数字のID
- **IPアドレス**：Supabaseのサーバーログに記録される可能性があります

## 2. 保存先と保存期間

- すべてのデータはSupabase（米国リージョン）のクラウドデータベースに保存されます。
- メッセージの保存期間は現時点では無期限ですが、将来的に自動削除機能を導入する場合は別途通知します。
- ユーザーは自分のメッセージを削除できません（Supabaseのダッシュボードから開発者が管理します）。

## 3. 利用目的

- チャット機能の提供（メッセージの送受信、ルーム管理）
- システムの保守・トラブルシューティング
- 不正利用の監視（スパム対策、禁止行為の調査）
- 利用状況の分析（匿名化された統計データとして）

## 4. 第三者提供

- 開発者は、法令に基づく場合を除き、個人を特定できる情報を第三者に提供しません。
- ただし、Supabaseなどのサービスプロバイダーへのデータ送信はこの限りではありません。
- URLの安全性チェックのため、送信されたURLはGoogle Safe Browsing APIに送信されます（Googleのプライバシーポリシーが適用されます）。

## 5. データの開示・訂正・削除

- 本プラグインの仕様上、一般ユーザーは自分のメッセージを直接削除できません。
- 削除を希望する場合は、開発者に問い合わせてください。ただし、開発者は技術的・運用上の理由から削除に応じられない場合があります。

## 6. 免責

開発者は、セキュリティ侵害やデータ漏洩が発生した場合でも、その結果生じるいかなる損害についても責任を負いかねます（故意または重大な過失がある場合を除く）。
